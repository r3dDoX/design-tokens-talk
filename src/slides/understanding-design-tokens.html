Introducing Design Tokens!

Design tokens are reusable, platform-agnostic, and context-independent variables that represent visual elements in a design system.

W3C Community Group: Github

Basic format


        {
            "color": {
                "brand": {
                    "brand-100": {
                        "description": "Brand color with 100% saturation",
                        "type": "color",
                        "value": "#ad8863ff",
                    },
                },
                "action": {
                    "primary": {
                        "value": "{color.brand.brand-100}"
                    }
                }
            }
        }
value is the only mandatory field values can be string, complex objects or references to other tokens
